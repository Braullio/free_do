# Free_Do

Create a simple TO DO LIST app (trello).
Duty is to be played running on AWS (Free Tier) with all code on Github.

#### Features:
  - Boards unlimited
  - Tasks unlimited
  - Etapas unlimited
 
#### Compulsory Technologies:
  - Ruby On Rails
  - Bootstrap
  - rspec, faker, factory_bot_rails
  - RDS Aurora
  - Elastic Beanstalk
